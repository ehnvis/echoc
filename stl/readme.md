* STL files for 3D printing

Print one of each file with following settings

Walls: 4
Layer height: 0.2mm
Nozzle: 0.4
Top layers: 4
Bottom layers: 4
Ironing: on
Infill: 20% any infill type

Exception is bottom that should be printed with
Walls: 4
Layer height: 0.2mm
Nozzle: 0.4
Top layers: 0
Bottom layers: 0
Ironing: Off
Infill: 40% hexagon
